# Analytics Sandbox *:)*

This repository is my personal learning space for Data Analytics.

I use it to upload practice work, case studies, notes, small tasks, and whatever I’m currently learning across SQL, Python, Excel, Power BI, and Statistics.

It is not a fixed project portfolio — it is more like a record of my learning journey.

Instead of keeping files scattered across folders, I wanted one place where I can document my progress and keep everything organized.

Some work here may be simple, some experimental, and some may later become full projects in separate repositories.

The goal is simple:

> Learn consistently, practice regularly, and keep improving.

This repository will continue to grow and change as I learn more.

Progress over perfection 🚀
